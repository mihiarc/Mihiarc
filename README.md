# Hi, I'm Chris Mihiar

Research Economist at the USDA Forest Service, Southern Research Station. I build open-source software tools that make environmental and natural resource research more accessible and reproducible.

My work sits at the intersection of economics, ecology, and data science — developing tools for forest inventory analysis, land-use modeling, climate data processing, and natural capital accounting. Key contributor to the [USDA Forest Service 2020 RPA Assessment](https://research.fs.usda.gov/understory/2020-resources-planning-act-assessment).

## Featured Projects

### pyfia
[![PyPI](https://img.shields.io/pypi/v/pyfia?color=blue)](https://pypi.org/project/pyfia/) [![Downloads](https://img.shields.io/pypi/dm/pyfia.svg)](https://pypi.org/project/pyfia/)

High-performance Python library for Forest Inventory and Analysis (FIA) data. Design-based estimation with proper variance calculation, DuckDB backend, and spatial filtering support.

### pyfvs
[![GitHub](https://img.shields.io/github/stars/mihiarc/pyfvs?style=social)](https://github.com/mihiarc/pyfvs)

Python implementation of the Forest Vegetation Simulator (FVS) for the Southern variant. Simulates growth and yield of southern yellow pine species.

### rpa-landuse
[![GitHub](https://img.shields.io/github/stars/mihiarc/rpa-landuse?style=social)](https://github.com/mihiarc/rpa-landuse)

Natural language query agent for RPA land use datasets. Query complex forest and economic data using conversational AI.

## Research Focus

- **Forest inventory & analysis** — Statistical methods for large-scale forest data
- **Land-use change modeling** — Projecting future scenarios under climate and socioeconomic conditions
- **Natural capital accounting** — Quantifying the value of forests' carbon storage and timber assets
- **Climate adaptation** — Tools for addressing wildfires, water scarcity, and extreme weather
